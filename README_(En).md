# ♾️ Universal Prompt Engine (UPE) v1.0

**Universal Prompt Engine (UPE)** is a **universal meta-prompt** created to generate prompts that are much clearer, more complete, and more effective for any LLM.

Instead of asking something directly to the AI, the UPE conducts a **structured interview** with the user to collect context, constraints, and goals before generating the final prompt.

The result is a prompt that is **more precise, smarter, and produces significantly better responses**.

---

# 🧠 What Is a Meta-Prompt

A **meta-prompt** is a prompt whose purpose is to **create other prompts**.

Instead of asking the AI directly, you first use the UPE to:

1. Explore the problem
2. Structure the context
3. Define constraints
4. Refine the objective

After that, the UPE generates an **optimized prompt** ready to be used in any LLM.

# ⚙️ How It Works

The UPE operates in three main steps:

### 1️⃣ Interview

The system asks the user questions to understand:

- goal
- context
- constraints
- desired output format

---

### 2️⃣ Identification of the Prompt Type

The UPE automatically identifies whether the case requires:

- **Master Prompt** (about the user)
- **Solution Prompt** (about a specific problem)

---

### 3️⃣ Generation of the Final Prompt

The UPE creates a complete prompt using modern prompt engineering techniques such as:

- **Act As (Role Definition)**
- **First Principles**
- **Deep Research**
- **Devil’s Advocate**
- **Constraints First**
- **Format As**
- **Verify & Cite**

---

# 🧭 Interview Modes

The user can choose the level of depth for the interview.

### Mode 1 — Simple

3 to 5 essential questions.

Ideal for quick tasks.

---

### Mode 2 — Consultative

6 to 10 structured questions.

Ideal for planning, projects, and decisions.

---

### Mode 3 — Deep

10+ exploratory questions.

Ideal for complex or strategic problems.

---

# 🚀 Quick Start

1. Copy the prompt from the **Universal Prompt Engine**.
2. Paste it into an LLM (e.g., ChatGPT).
3. Answer the questions asked by the system.
4. When you finish the interview type: `Done`
5. The UPE will generate an **optimized final prompt**.

---
# 💡 Example

Original problem: _"I want to make money"_

After using the UPE: _"I have X reais available, I work with Y, I am interested in Z and I want to generate additional income within up to 3 months using digital skills."_

This level of context allows the LLM to produce **much more useful and specific responses**.
